building

object building

Das Objekt building kann simuliert oder mit realer Hardware verwendet werden.

Defaultmaessig wird das Gebaeude simuliert. Es kann mit der Methode hardwareOn() die reale Gebaeude-Hardware angesprochen werden.

hardwareOn
Link copied to clipboard
fun hardwareOn(port: String = "None", timeOutSeconds: Int = 10)

Wechselt von der Simulation zur realen Hardware. Nach erfolgreicher Verbindung erscheint "Connection Ready" auf der Konsole und auf dem Display der Hardware. Ohne Angabe des Port-Parameter wird nach dem COM-Port gesucht. Mit Angabe des Port-Parameter wird der gewünschte COM-Port geöffnet. Formatbeispiel des Portnamens: Windows: COM3, Linux: /dev/tty3.
